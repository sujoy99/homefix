# HomeFix — Web App

> Next.js 15 | App Router | SSR/SSG

## Setup

```bash
# From monorepo root
npm install
npm run web:dev
```

## Tech Stack

- Next.js 15 (App Router)
- Zustand (State Management)
- Axios + TanStack Query (API Layer)
- Vanilla CSS + CSS Modules
- next-intl (Bengali + English)
